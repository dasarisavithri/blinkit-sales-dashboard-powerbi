# blinkit-sales-dashboard-powerbi
Interactive Power BI dashboard analyzing Blinkit sales performance, customer behavior, outlet performance, and key business insights using DAX and Power Query.

## Project Overview

This Power BI dashboard provides a comprehensive analysis of Blinkit's sales performance, customer purchasing behavior, product categories, and key business metrics.

## Business Problem

The objective of this project is to analyze sales data and identify trends that can help improve business performance, inventory planning, and customer satisfaction.

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Key Performance Indicators

* Total Sales
* Average Sales
* Number of Orders
* Customer Ratings
* Product Category Performance

## Dashboard Features

* Interactive Filters and Slicers
* Sales Trend Analysis
* Outlet Performance Analysis
* Product Category Insights
* Customer Rating Analysis

## Key Insights

* Identified top-performing product categories.
* Compared outlet performance across locations.
* Analyzed customer ratings and purchasing patterns.
* Evaluated overall sales trends and business growth opportunities.
